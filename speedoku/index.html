How to Play

Speedoku is competitive real-time Sudoku. You and your opponents solve the same puzzle simultaneously.

  • Fill the 9×9 grid so every row, column, and 3×3 box contains each digit 1–9 exactly once.
  • Click a cell, then press a number key or use the number pad.
  • You can see your opponents' boards — not the numbers, but how fast they're filling up.
  • First to complete the grid wins in ranked mode.
  • In elimination mode, wrong answers cost lives. Lose all lives and you're out.
  • Toggle notes mode (✏) to pencil in candidate numbers.

Keyboard: Arrow keys to navigate · 1–9 to enter · Backspace/Delete to erase · N to toggle notes
